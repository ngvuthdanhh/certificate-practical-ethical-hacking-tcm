# Practical Ethical Hacking – TCM

This repository contains **notes, labs, workshops, projects, extras, and certificate** for the *Practical Ethical Hacking* course offered by TCM.  

It provides a structured learning path for mastering **ethical hacking techniques, penetration testing, and defensive strategies**.  

## Structure

- `notes/` – Detailed theoretical notes for each module  
- `labs/` – Hands-on exercises and practical labs  
- `workshops/` – Red and blue team exercises, scenario-based learning  
- `projects/` – Realistic penetration testing and security assessment projects  
- `extras/` – Case studies, learning resources, and timeline  
- `docs/` – Glossary, syllabus, roadmap, references  
- `cert/` – Certificate of completion  
- `screenshots/` – Visual snapshots of course materials

This file contains real-world case studies to understand practical attacks and defenses.

## Case Study 1: Corporate Network Breach
- Attack Vector: Phishing email leading to credential theft  
- Exploit Used: Remote Desktop compromise  
- Outcome: Sensitive data exfiltrated  
- Mitigation: Multi-factor authentication, employee training, monitoring  

## Case Study 2: Web Application Exploit
- Attack Vector: SQL Injection on customer portal  
- Exploit Used: Unauthorized database access  
- Outcome: Data leakage of customer information  
- Mitigation: Input validation, parameterized queries, WAF deployment  

## Case Study 3: Wireless Network Attack
- Attack Vector: WPA2 cracking using weak passwords  
- Exploit Used: MITM attack on internal Wi-Fi  
- Outcome: Access to internal systems  
- Mitigation: Strong passwords, WPA3 adoption, network segmentation

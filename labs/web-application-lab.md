# Web Application Hacking Lab

## Objective
Identify and exploit web application vulnerabilities safely.

## Tools
- Burp Suite
- OWASP ZAP
- DVWA or other lab web apps

## Exercises
1. Test for SQL Injection vulnerabilities.
2. Exploit XSS and CSRF vulnerabilities.
3. Implement security fixes in a lab environment.
4. Document vulnerabilities and mitigation.

## Safety
Use only controlled lab web applications.

# Sniffing Lab

## Objective
Capture and analyze network traffic.

## Tools
- Wireshark
- tcpdump
- Virtual lab network

## Exercises
1. Capture packets from a lab network.
2. Filter HTTP, FTP, and SMTP traffic.
3. Identify credentials or sensitive data.
4. Implement countermeasures in lab environment.

## Safety
Do not capture traffic from unauthorized networks.

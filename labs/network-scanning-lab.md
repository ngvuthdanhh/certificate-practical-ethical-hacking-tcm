# Network Scanning Lab

## Objective
Learn how to discover live hosts, open ports, and services in a network.

## Tools
- Nmap
- Netcat
- Zenmap (GUI for Nmap)

## Exercises
1. Perform a ping sweep on a target subnet.
2. Scan for open TCP and UDP ports.
3. Identify services and versions running on open ports.
4. Document findings for analysis.

## Safety
Perform all scans in a controlled lab environment.

# Enumeration Lab

## Objective
Extract detailed information about users, shares, and services.

## Tools
- Nmap scripts
- SNMPwalk
- LDAP enumeration tools

## Exercises
1. Enumerate user accounts on a target system.
2. Identify shared resources on network hosts.
3. Discover services and banner information.
4. Record all findings securely.

## Safety
Use isolated lab targets only.

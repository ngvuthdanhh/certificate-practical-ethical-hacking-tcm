# System Hacking Lab

## Objective
Practice exploiting vulnerabilities to gain access and escalate privileges.

## Tools
- Metasploit Framework
- Hydra / John the Ripper
- Vulnerable lab machines (e.g., Metasploitable)

## Exercises
1. Exploit weak passwords to gain initial access.
2. Perform privilege escalation.
3. Maintain access safely in the lab.
4. Document the methodology and mitigation steps.

## Safety
Do not target external networks.
